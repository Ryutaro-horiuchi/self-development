# 4.17.1 DOM Based XSS

## 概要

- サーバー側のプログラム不備だけではなく、JSによる処理の不備が原因でXSSとなる場合がある
    
    → `DOM Based XSS` と呼ばれている
    

## 攻撃手法と影響

### innerHTMLによるDOM Based CSS

- 前提
    - URLの#以下の部分(フラグメント識別子やハッシュなどと呼ばれる)に応じて表示変化させるアプリケーション
    
    ```jsx
    <body>
    <script>
    window.addEventListener("hashchange", chghash, false);
    window.addEventListener("load", chghash, false);
    
    function chghash() {
      var hash = window.location.hash;
      var color = document.getElementById("color");
      color.innerHTML = decodeURIComponent(window.location.hash.slice(1));
    }
    </script>
    <a href="#赤">赤</a>
    <a href="#緑">緑</a>
    <a href="#青">青</a>
    <p id="color">赤</p>
    
    </body>
    ```
    
- 攻撃
    
    ```
    http://example.jp/4h/4h-001.html#%3Cimg%20src=/%20onerror=alert(1)%3E
    ```
    
    → 1がalertで表示される
