## 再帰関数

- 手続きの中で自分自身を呼び出すことを再起呼び出しという。再起呼び出しを行う関数のことを再帰関数と呼ぶ
- イメージ
    
    ```python
    def rec(N):
    	# 終了条件
    	if N == 0:
    		return 0
    
    	# 再帰呼び出し
    	return N + rec(N-1)
    ```
    
    - 再帰呼び出しを行わずに直接 return する「終了条件」を先に記述する

### [例題 ABC115  「**D - Christmas**」](https://atcoder.jp/contests/abc115/tasks/abc115_d)

- input: N がレベル, X が食べ進めた層の数

```python
def rec(N, X, L, S):
    """バティの数を返す関数

    Args:
        N: レベル
        X: 食べ勧めた層の数
        L: バーガーの長さ 配列
        S: パティの数 配列
    Returns:
        食べたパティの数
    """

    if N == 0:
        return 1

    if X == 1:
        return 0
    elif X <= L[N-1] + 1:
        return rec(N-1, X-1, L, S)
    elif X == L[N-1] + 2:
        return S[N-1] + 1
    elif X <= 2 * L[N-1] + 2:
        return rec(N-1, X  - L[N-1] -2, L, S) + S[N-1] + 1
    else:
        return 2 * S[N-1] + 1

N, X = map(int, input().split())
L = [1] * (N + 1) # 2 * L[N-1] + 3
S = [1] * (N + 1) # 2 * S[N-1] + 1

for i in range(1, N + 1):
    L[i] = 2 * L[N-1] + 3
    S[i] = 2 * S[N-1] + 1

print(rec(N, X, L, S))

```

### 再帰関数を用いた全探索

- 例題 「ナップザック問題」ᴺᴺ
    
    ![image.png](attachment:88d1adcd-c798-422e-9ba5-6bb7bb0f667d:image.png)
    
    - 各品物を選ばないを0, 選ぶを1とした時、0と1からなる 長さ N の数列をすべて列挙することで、ナップザック問題が解けるようになる
    - 動的にNが変化するため、事前にfor文を定義することはできない。このような時に再帰関数を用いる
    - 再帰関数を用いて、2ᴺ通りの組み合わせを全探索する
        
        ```python
        # 数列AのサイズNを仮に 3 としておく
        N = 3
        
        # 0, 1からなる長さNの数列Aを列
        def rec(A):
            if len(A) == N:
                print(A)
                return
        
            for v in range(2):
                A.append(v)
                rec(A) # A に要素を追加した上で再帰的に呼び出す
                A.pop() # Aを元に戻す
        
        rec([])
        ```
        
- [例題 ABC165 「C - Many Requirements」](https://atcoder.jp/contests/abc165/tasks/abc165_c)
    
    ```python
    N, M, Q = map(int, input().split()) # 3,4,3
    
    a = [0] * Q
    b = [0] * Q
    c = [0] * Q
    d = [0] * Q
    for q in range(Q):
        a[q], b[q], c[q], d[q] = map(int, input().split())
    
        # 数列Aの添字 a[q], b[q] を0から始まるようにする
        a[q] -= 1
        b[q] -= 1
    
    # 数列Aのスコアを計算する関数
    def calc_score(A):
        score = 0
        for ai, bi, ci, di in zip(a,b,c,d):
            if A[bi] - A[ai] == ci:
                score += di
        return score
    
    def rec(A):
        # 終了条件
        if len(A) == N:
            return calc_score(A)
    
        # 最大値
        result = 0
    
        # 数列 A の前回の要素を取得
        prev_last = A[-1] if A else 1
    
        # 数列 A が単調増加になるように末尾に要素を追加
        for add in range(prev_last, M + 1):
            A.append(add)
    
            result = max(result, rec(A))
    
            A.pop()
    
        # 最大値を返す
        return result
    
    A = []
    print(rec(A))
    ```
    

## 幅優先探索

グラフの始点に近いところから順に探索するアルゴリズム

- 参考
    
    [幅優先探索](https://www.notion.so/1e493db05554808eb135d958f90868fe?pvs=21) 
    

### 幅優先探索で用いるデータ

| 変数名 | データの型 | データの役割 |
| --- | --- | --- |
| que | キュー | これから探索する予定の頂点を格納する |
| dist | 配列 | 探索終了時において、dist[v]は、始点から頂点 v への最短経路の長さを表す |

### 幅優先探索コード例

```python
from queue import Queue

def bfs(G, s):
    """グラフGに対して、頂点sからの幅優先探索を行う関数
    戻り値は、sから各頂点への距離を格納したリスト
    """

    que = Queue()

    dist = [-1] * len(G)  # 距離を格納するリスト

    que.put(s) # 始点をキューに追加
    dist[s] = 0 # 始点の距離は0

    while not que.empty():
        v = que.get() # キューから頂点を取り出す

        for v2 in G[v]: # 頂点vに隣接する頂点iについて
            if dist[v2] != -1: # すでに訪問済みの頂点はスキップ
                continue

            que.put(v2) # 頂点iをキューに追加
            dist[v2] = dist[v] + 1 # 頂点iの距離を更新

    return dist
```

## 動的計画法

- 元の問題を各部分問題に分割し、それぞれの解を求めながら順次求めていく手法を動的計画法と呼ぶ
- コード例
    
    ```python
    INF = 2 ** 60
    N = int(input())
    h = list(map(int, input().split()))
    
    dp = [INF] * N
    dp[0] = 0
    
    # 足場 1, 2, ..., N-1 への最小コストを順に求める
    for i in range(1, N):
        if i - 1 >= 0:
            dp[i] = min(dp[i],dp[i-1] + abs(h[i] - h[i-1]))
        if i - 2 >= 0:
            dp[i] = min(dp[i],dp[i-2] + abs(h[i] - h[i-2]))
    
    print(dp[N-1])
    ```

## 貧欲法

- 各ステップで、その時点での「最善の選択（局所最適解）」を貪欲に選び続ける手法
- 将来のことを考えず、今この瞬間に最もよく見える選択を繰り返すことで、問題全体の解を求める
- 例 「コイン問題」
    - 「500円・100円・50円・10円を使って370円を最少枚数で払う」
    → 常に最大のコインから選ぶ → 100×3 + 50×1 + 10×2 = 6枚
- 動的計画法との比較
    
    
    |  | 貧欲法 | 動的計画法 |
    | --- | --- | --- |
    | アプローチ | 局所最適を積み重ねる | 全パターンを記録・比較 |
    | 速度 | 速い | 比較的遅い |
    | 最適性保証 | 問題による | 常に最適解 |
    |  |  |  |

### 例題「[ABC 137: D Summer Vacation](https://atcoder.jp/contests/abc137/tasks/abc137_d)」

- heapを使用する
    - heapとは
        - 特定の順序規則を持つ木構造のデータ構造
            - ヒープは完全二分木（二分木をできるだけ左から詰めたもの）で表される
            
            ```python
                    1   ← 根（最小値）
                   / \
                  3   2
                 / \ / \
                7  4 5  6
            ```
            
    - 配列で表現できる
        
        ```python
        インデックス:  0  1  2  3  4  5  6
        配列:         [1, 3, 2, 7, 4, 5, 6]
        ```
        
        - 左の子： 2i + 1
        - 右の子： 2i + 2
        - 親：     (i - 1) // 2
        - heappush
            - 末尾に要素を追加
            - 親と比較しながら上に移動（ふるい上げ）
            
            ```python
            heap = []
            heapq.heappush(heap, 3)
            heapq.heappush(heap, 1)
            heapq.heappush(heap, 2)
            print(heap)  # [1, 3, 2]（内部的にヒープ順序を維持）
            ```
            
        - heappop
            - 根（最小値）を取り出す
            - 末尾の要素を根に移動
            - 子と比較しながら下に移動（ふるい下げ）
            
            ```python
            print(heapq.heappop(heap))  # 1（常に最小値が返る）
            print(heapq.heappop(heap))  # 2
            print(heapq.heappop(heap))  # 3
            ```
            
    - 計算量
        
        
        | 操作 | 計算量 |
        | --- | --- |
        | `heappush` | O(log n) |
        | `heappop` | O(log n) |
        | 最小値参照 `heap[0]` | O(1) |
    - 「最小値（最大値）を素早く取り出したい」場面に最適！
        
        Pythonのheapqは最小ヒープのみ対応。最大ヒープを使いたい場合は値を負にするのが定番。
        
        ```python
        heap = []
        heapq.heappush(heap, -3)
        heapq.heappush(heap, -1)
        heapq.heappush(heap, -2)
        
        max_val = -heapq.heappop(heap)
        print(max_val)  # 3（最大値）
        ```
        
- 解
    
    ```python
    from heapq import heappush, heappop
    
    N, M = map(int, input().split())
    
    AtoB = [[] for i in range(M+1)]
    
    for i in range(N):
        A, B = map(int, input().split())
        if A > M:
            continue
    
        AtoB[A].append(B)
    
    result = 0
    
    # ヒープ
    que = []
    
    print(f"AtoB: {AtoB}")
    print("#"*10)
    for Bs in AtoB:
        for B in Bs:
            heappush(que, -B)
            print(f"for B in Bs B: {B}")
            print(que)
        
        if que:
            result -= heappop(que)
    
    print(result)
    
    ```
