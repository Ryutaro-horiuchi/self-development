## 再帰関数

- 手続きの中で自分自身を呼び出すことを再起呼び出しという。再起呼び出しを行う関数のことを再帰関数と呼ぶ
- イメージ
    
    ```python
    def rec(N):
    	# 終了条件
    	if N == 0:
    		return 0
    
    	# 再帰呼び出し
    	return N + rec(N-1)
    ```
    
    - 再帰呼び出しを行わずに直接 return する「終了条件」を先に記述する

### [例題 ABC115  「**D - Christmas**」](https://atcoder.jp/contests/abc115/tasks/abc115_d)

- input: N がレベル, X が食べ進めた層の数

```python
def rec(N, X, L, S):
    """バティの数を返す関数

    Args:
        N: レベル
        X: 食べ勧めた層の数
        L: バーガーの長さ 配列
        S: パティの数 配列
    Returns:
        食べたパティの数
    """

    if N == 0:
        return 1

    if X == 1:
        return 0
    elif X <= L[N-1] + 1:
        return rec(N-1, X-1, L, S)
    elif X == L[N-1] + 2:
        return S[N-1] + 1
    elif X <= 2 * L[N-1] + 2:
        return rec(N-1, X  - L[N-1] -2, L, S) + S[N-1] + 1
    else:
        return 2 * S[N-1] + 1

N, X = map(int, input().split())
L = [1] * (N + 1) # 2 * L[N-1] + 3
S = [1] * (N + 1) # 2 * S[N-1] + 1

for i in range(1, N + 1):
    L[i] = 2 * L[N-1] + 3
    S[i] = 2 * S[N-1] + 1

print(rec(N, X, L, S))

```
