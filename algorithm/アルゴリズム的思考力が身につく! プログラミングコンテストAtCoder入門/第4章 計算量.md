## 計算量を学ぶ意義

- C問題からはアルゴリズムの計算量を意識する必要が生まれる
- 多くの問題で制限時間は「2秒」であり、この制限内で効率的なアルゴリズムを選択する必要がある

## 計算量の使い方

- 制限時間2秒に対する限度
    - O(N)の計算量を持つ記法は、N ≤  10⁸が限度
    - O(N²)の計算量を持つ記法は、N ≤ 10⁴が限度
- AtCoderではよく見られる制限は N ≤ 10⁵であるため、O(N²)  ではTLEになるため、O(N)やO(N log N)への高速化が必要になる
