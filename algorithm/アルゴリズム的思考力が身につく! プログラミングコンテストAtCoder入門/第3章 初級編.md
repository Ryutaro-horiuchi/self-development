## 各桁の和を求める問題

- 参考: [AtCoder ABC083 B問題「**Some Sums**」](https://atcoder.jp/contests/abc083/tasks/abc083_b)
- 正の整数 n が与えられあた時に、その各桁の和を求める方法は下記のように実装できる
    
    ```python
    def calc_sum_digits(n):
        """整数nの各桁の和を計算して返す関数"""
        sum_digit = 0
        while n > 0:
            sum_digit += n % 10
            n //= 10
        return sum_digit
    ```
    
    - n = 4649の場合
        1. n % 10をすると、9となる。これは整数1の位を表す
        2. 次に、`n //= 10 (n = n // 10)`を実行することで、nを10で割った時の商に更新する。具体的にはn = 464となる
        - 1 → 2をnが0になるまで繰り返す

### Tips: Python 組み込み関数allやlistの内包表記

- before
    
    ```python
    cnt = 0
    while True:
        can_do = True
        for i in A:
            if i % 2 != 0:
                can_do = False
        if not can_do:
            break
    
        A = list(map(lambda i: i / 2, A))
        cnt += 1
    ```
    
- after
    
    ```python
    cnt = 0
    while all(i % 2 == 0 for i in A):
    	A = [a // 2 for a in A]
    	cnt += 1
    
    ```

## 文字列

- 参考: [AtCoder ABC104 B問題「AcCepted」](https://atcoder.jp/contests/abc104/tasks/abc104_b)

```python
...

def isAC(S):
    if S[0] != 'A':
        return False

		# Sの先頭2文字と末尾1文字を除いた "C" の個数を調べる
    if S[2:-1].count("C") != 1:
        return False

		# Sに含まれる大文字の個数を調べる
    if sum(map(str.isupper, S)) != 2:
        return False

    return True

...
```

- `count("C")`
    - 文字列やリスト、タプルといったシーケンス型のデータ内で指定した要素や部分文字列が何回出現したかを数える
- `sum(map(str.isupper, S))`
    - mapは各文字に対して、isupperを適用する
        
        ```python
        # 中身
        ["A", "b", "C"] → [True, False, True]
        
        ```
        
    - sum関数
        - Pythonでは、Trueは1, Falseは0で数える
            - pythonでは、bool型はint型のサブクラス
