## 基本的な演算と記号

### 剰余(mod)

aをbで割った余りのことを a mod bと書く

```
8 mod 5 = 3 (8 ÷ 5 は 1余り3)

python
8 % 5
```

### 絶対値(abs)

```
|-20| = 20

python
abs(-20)
```

### 累乗(pow)

aをb回掛けた値をaのb乗と呼ぶ

```
10の2乗 = 10 × 10

python
10 ** 2
```

### ルート(sqrt)

- 0以上の実数aに対し、xの2乗 = aとなるような非負の実数xをルートaと呼び、√aと書く
- 言い換えると面積aの正方形の一辺の長さが√a

```
√4 = 2
√9 = 3
√1.96 = 1.4
√2 = 1.414213

python
import math
math.sprt(a)
```

- 有理数で表せないことがある

## 論理演算

0(FALSE) 1(TRUE)をとる値の間で行われる演算

- a AND b
    - a,bの両方が1であれば1、そうでなければ0
- a OR b
    - a,bの少なくとも片方が1であれば1、そうでなければ0
- a XOR b
    - a,bのうち一方のどちらかが1であれば1、そうでなければ0

## ビット演算

コンピューターで行われる演算の一つ

### ビット演算の流れ

1. 整数(通常は10進法)を2進法に変換する
2. 桁(1の位、2の位、4の位、8の位…)ごとに論理演算を行う
3. 論理演算の結果を整数(通常は10進法)に変換する

### AND演算

2進法表現の格桁ごとに論理演算のANDを行うもの。

- Ex. 11 AND 14
    - 11を2進法に変換すると「1011」
    - 14を2進法に変換すると 「1110」
    - 1の位に対するAND : 1 AND 0 = 0である
    - 2の位に対する AND : 1 AND 1 = 1である
    - 4の位に対する AND : 0 AND 1 = 0である
    - 8の位に対する AND : 1 AND 1 = 1である
    - 計算結果を上の結果からまとめていくと、「1010」であり、これを10進法に変換すると10である
    - よって、 11 AND 14 = 10である

### ビット演算の実装

プログラミング言語上では、ビット演算を行うために10進法から2進法に変換する必要はない

Pythonでは下記の演算子を使うことで実装できる

| AND | OR | XOR |
| --- | --- | --- |
| & | | | ^ |

### 3つの整数に対する演算の場合

- まず2つの値に対してビット演算を行い、次にその結果と残り1つの値に対してビット演算をお行うという流れが計算される
- どのような順序で計算しても結果は変わらない
- 3つ以上の整数に対する AND OR XOR演算を行うとき、各桁について下記の性質が成り立つ
    - AND演算 - すべての数についてその桁が1であれば、計算結果は1。そうでなければ0
    - OR演算 - どれか一つの数についてその桁が1であれば、計算結果は1。そうでなければ0
    - XOR演算 - その桁が1である数が奇数個あれば、計算結果は1。そうでなければ0

### 左シフトと右シフト

- 整数a,bに対する演算であり、aを2進法で表した時のビットをb個左 / 右にずらすもの
- プログラング言語上のデータ型ごとに2進法で表記できる桁が決まっているため、シフト演算を行うと「溢れたビット」が発生する。これらは消されることに注意する
- Pythonでは、左シフトは `a << b`。右シフトは `a >> b`で実装できる
    - Ex. 46 << 1 = 92、46 >> 2 = 11
- (1 << N)は2のN乗を表し、組み合わせの全探索や繰り返し二乗法などで使う

## いろいろな関数

### 関数とは

- 入力が決まると出力の値が一つに決まる関係のこと
- Ex. 入力した数の2乗を返す関数
    - `y = x²`
        - 入力をxと表し、出てくる数をyとする
        - yの代わりに、`f(x) = x²`のように書くこともある
            - 具体的な数値を入れた形式で書くこともあり、`f(10) = 100`, `f(17) = 289`となる

### 座標平面とは

- 点の位置を座標で表す平面のこと
- 横方向x軸、縦方向y軸とする
- 原点から右方向にa, 上方向にb進んだ点を座標(a,b)とする

### 一次関数

- `y = ax + b`の形式で表される、以下のような関数を一次関数という
    
    ```
    y = x + 1
    y = 3x
    y = 314x - 159
    ```
    
- グラフ

    - xの値が1増えると必ずyの値がaだけ増え、このことを「1次関数の傾きがaである」という

### 二次関数

- `y = ax² + bx + c`の形で表される、以下のような関数を一次関数と呼ぶ
    
    ```
    y = x²

    y = 0.1x²
    y = -31x² + 41x - 59
    ```

### 多項式と多項式関数

- 多項式関数
    - 一次関数はxまで、二次関数はx²まで出現したが、x³以上にも範囲を広げた関数のこと(一次関数, 二次関数も含まれる)
- 多項式
    - `y =` の部分を取り去り、文字式の形で表したものを多項式と呼ぶ
    - 単項式

### 項

- 項
    - 多項式を構成する一つ一つの単項式のこと

    - それぞれの項から数だけを取り出したものを係数と呼ぶ

### 対数関数

- 対数log𝞪 bは指数をひっくり返したもので、αを何乗したらbになるかを表す
- Ex.
    - 10を3乗すると、10 × 10 × 10 = 1000になるので、`log₁₀ 1000 = 3`
    - 2⁰ = 1なので、`log₂ 1 = 0`
    - 2¹ = 2なので、`log₂ 2 = 1`
    - 2² = ４なので、`log₂ 4 = 2`
- log𝞪 bの𝞪を底、bを真数と呼ぶ。
    - 底が1以外の正の数であり、真数が正の数である場合にしか、対数を計算することはできない
    - 底が10であるような対数 `log₁₀ x` を常用対数と呼び、xを10進法で表した時のおおよその桁数を示す
    - 底が2であるような対数 `log₂ x`は二分探索法の計算回数など、アルゴリズムの文脈で頻出
- `y = log𝞪 b`の形で表される関数を対数関数と呼ぶ

    - xの値が増えてもyの値が増加しない特徴がある
